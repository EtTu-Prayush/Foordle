## Installations

### Email JS

* For Linux in your frontend:
```
npm i emailjs-com
```

###  Chart MUI

*For Linux in your frontend:
```
npm i @devexpress/dx-react-chart-material-ui
```

### Running the code

* Run Mongo daemon:
```
sudo mongod
```
Mongo will be running on port 27017.


* Run Express Backend:
```
cd backend/
npm install
npm start
```

* Run React Frontend:
```
cd frontend
npm install/
npm start
```

Navigate to [http://localhost:3000/](http://localhost:3000/) in your browser.
